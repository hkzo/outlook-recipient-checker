# 概要

* 本リポジトリのファイル群は、Outlook Add-inの**ビルド後成果物（静的ファイル）**です
* ソースコードは含まれておらず、デプロイ後の配布物として利用されます
* GitHub Pages によりホスティングされ、公開用成果物を配信します

# 公開URL

GitHub Pages により以下のURLで公開されます：

```text
https://hkzo.github.io/outlook-recipient-checker/manifest.xml
```

# 注意事項

* 本リポジトリは**ビルド成果物専用**です
* 本リポジトリで直接編集した内容は、次回デプロイ時に上書きされる可能性があります
* 変更を行う場合は、管理元で修正し、再ビルド後の成果物を反映してください

## デプロイについて

- 本リポジトリはGitHub Actionsにより自動更新されます
- 手動でのデプロイ作業は不要です
